# Source code

Core MATLAB routines for solving the Smoluchowski equation under shear flow using intermediate asymptotics.

## Components

The source code includes:

- hydrodynamic interaction functions
- lubrication-force models
- inner and outer asymptotic solutions
- patching procedures
- numerical integration routines
- interaction potentials
- radial distribution function solvers

These routines are used by the example scripts contained in the `examples/` folder.
